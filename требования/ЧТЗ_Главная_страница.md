# ЧТЗ: Разработка главной страницы сайта Dry Cleaning D&A

**Дата:** 2026-04-20
**Статус:** Утверждён
**Маршрутизация:** Разработчик (frontend)
**Архитектор:** Не требуется

---

## 1. Описание задачи

Разработка главной страницы (landing page) сайта da-drycleaning.ru на Next.js + Tailwind CSS по дизайн-брифу. 13 секций, полная мобильная адаптация, анимации, интеграция с FastAPI бэкендом.

## 2. Технологический стек

| Компонент | Технология |
|---|---|
| Фреймворк | Next.js 15 (App Router, TypeScript) |
| Стили | Tailwind CSS 4 |
| Шрифты | Montserrat (заголовки), Inter (текст) — Google Fonts |
| Иконки | Lucide React |
| API-клиент | fetch с типизацией (встроенный Next.js) |
| Контейнеризация | Docker + nginx |
| CI/CD | GitHub Actions |

## 4. Секции главной страницы (13 шт.)

### TASK-FRT-001: TopBar (Section 1)
- Фон: #0A1F44, текст белый, высота 40px
- Контент: город, email, график, рейтинг Яндекс, мессенджеры
- Мобильная: скрыть email, оставить телефон + мессенджеры

### TASK-FRT-002: Navigation (Section 2)
- Sticky, высота 70px, фон белый, тень при скролле
- Логотип слева, меню по центру, телефон + CTA-кнопка справа
- Мобильная: гамбургер → полноэкранное меню с CTA внизу

### TASK-FRT-003: Hero (Section 3)
- Полноширинный блок 560px
- H1: «Профессиональная выездная химчистка в Москве и МО»
- Подзаголовок + поле адреса + кнопка «Вызвать мастера»
- Рейтинг, количество заказов, «Оплата по факту»
- Плейсхолдер-фон

### TASK-FRT-004: Услуги — 6 карточек (Section 4)
- Фон белый, сетка 3x2 (десктоп), 2 колонки (мобильный)
- Карточки: фото, название, описание, «Подробнее →»
- Hover: translateY(-4px) + box-shadow
- 6 из 12 услуг: Диваны, Ковры, Матрасы, Авто, Шторы, Ковролин
- Кнопка «Все 12 услуг →»

### TASK-FRT-005: Как мы работаем — 3 шага (Section 5)
- Фон #F7F8FA
- 3 шага: Заявка → Мастер приедет → Примите результат
- Иконка + номер + заголовок + описание
- Соединяющие линии/стрелки
- Мобильная: вертикальный стек

### TASK-FRT-006: Before/After (Section 6)
- Интерактивный ползунок сравнения
- Табы: Диваны, Ковры, Матрасы, Автомобили
- Плейсхолдеры для фото
- Подпись: «Переместите ползунок, чтобы увидеть разницу»

### TASK-FRT-007: Почему выбирают нас (Section 7)
- 4 карточки: Гарантия, Безопасность, Без предоплаты, Работаем ежедневно
- Иконки линейные, цвет #3399CC
- 2x2 на мобильном

### TASK-FRT-008: Акции — 2 баннера (Section 8)
- «Приведи друга — скидка 20%» (градиентный фон)
- «Пенсионерам — скидка 10%» (приглушённый фон)
- Мобильная: вертикальный стек

### TASK-FRT-009: Организациям/B2B (Section 9)
- Фон #F7F8FA
- 4 карточки: Офисы, Рестораны, Школы, Дет. сады
- CTA: «Получить коммерческое предложение»

### TASK-FRT-010: Отзывы (Section 10)
- Карусель (мобильный) / сетка (десктоп)
- Карточки с рейтингом ★★★★★, текстом, автором
- Ссылка «Все отзывы» + «Рейтинг на Яндекс Картах»

### TASK-FRT-011: Форма заявки/CTA (Section 11)
- Фон #0A1F44, белый текст
- Inline форма: Имя + Телефон + Тип услуги + Отправить
- Мобильная: stacked
- Телефон как альтернатива

### TASK-FRT-012: FAQ — аккордеон (Section 12)
- 6 вопросов с плавным раскрытием
- Иконка +/−

### TASK-FRT-013: Footer (Section 13)
- Фон #0A1F44
- 4 колонки: Логотип, Услуги, Компания, Контакты
- Копирайт

### TASK-FRT-014: API-слой
- Типизированный fetch-клиент для FastAPI сервисов
- Endpoint: Catalog Service (:8003) — список услуг
- Endpoint: Review Service (:8009) — отзывы
- Endpoint: Order Service (:8002) — создание заявки из формы
- Fallback на моковые данные при недоступности API

### TASK-FRT-015: Анимации и UX
- Секции: fade-in + translateY при скролле
- Кнопки: translateY(-1px) + box-shadow при hover (300ms)
- Карточки: translateY(-4px) при hover (400ms)
- Навигация: тень при скролле
- FAQ: плавное раскрытие (300ms)
- Мобильное меню: slide-in

## 5. Инфраструктура

### TASK-INF-001: Docker
- Dockerfile для Next.js (multi-stage build)
- docker-compose.dev.yml с nginx для фронтенда
- nginx.conf для проксирования API к FastAPI gateway

### TASK-INF-002: CI/CD (GitHub Actions)
- lint + type-check при push
- build при merge в main
- Docker build + push при теге

## 6. Критерии приёмки

1. Все 13 секций рендерятся корректно на десктопе (1920px, 1200px)
2. Все 13 секций адаптированы для мобильных (375px, 768px)
3. Все CTA-кнопки кликабельны и ведут на форму заявки
4. FAQ-аккордеон раскрывается/закрывается плавно
5. Before/After ползунок работает на десктопе и мобильном (touch)
6. Навигация sticky с тенью при скролле
7. Мобильное меню открывается/закрывается корректно
8. Форма заявки валидирует поля перед отправкой
9. API-слой корректно работает с FastAPI (с fallback на моки)
10. Docker-контейнер собирается и запускается
11. ESLint + TypeScript компиляция без ошибок
12. Lighthouse Performance > 80, Accessibility > 90
